Abstract base class representing a generic numeric progression.

This class defines the common interface for all progressions. It cannot be instantiated directly because \doxylink{classProgression_aaf7ff9664dc26722c4839b9bcccf219f}{next\+\_\+value()} is pure virtual. Derived classes must implement their specific way of advancing to the next term.
