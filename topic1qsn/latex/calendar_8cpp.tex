\doxysubsection{Detailed Description}
A C++23 console application to generate a yearly calendar. 

This program accepts a four-\/digit year and prints a formatted calendar for that year to the standard output. It handles leap year logic, calculates the starting day of the week automatically, and formats the output using {\ttfamily std\+::print} and ANSI color codes.
\begin{DoxyItemize}
\item \begin{DoxyNote}{Note}
This program requires a C++23 compliant compiler due to the usage of {\ttfamily \texorpdfstring{$<$}{<}print\texorpdfstring{$>$}{>}} and {\ttfamily std\+::format}. 
\end{DoxyNote}

\end{DoxyItemize}
