A C++23 console application to generate a yearly calendar.

Contents

  • Reference
    • Functions
    • Variables

This program accepts a four-digit year and prints a formatted calendar for that year to the standard output. It handles leap year logic, calculates the starting day of the week automatically, and formats the output using std::print and ANSI color codes.

  • Note

    This program requires a C++23 compliant compiler due to the usage of <print> and std::format.

int calculate_jan_1_day(int year)

Calculates the day of the week for January 1st of the given year.

Parameters
year The four-digit year.
Returns An integer representing the day of the week (0 = Sunday, 1 = Monday, ..., 6 = Saturday).

This function uses a variation of the standard day-of-week algorithms (similar to Gauss's or Zeller's) to determine the offset.

int days_in_feb(int year)

bool is_leap_year(int year)

Checks if a specific year is a leap year.

Parameters
year The year to validate.
Returns true if the year is a leap year, false otherwise.

A year is a leap year if it is divisible by 4, except for end-of-century years, which must be divisible by 400.
